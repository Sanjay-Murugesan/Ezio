# 🚀 Ezio Automation Website

A responsive industrial automation business website with enquiry management, EmailJS integration, and admin dashboard functionality.

## 🌐 Live Demo

🔗 https://ezio-automation.netlify.app/

---

## 📌 Features

### Main Website
- Responsive industrial UI
- Contact & enquiry form
- EmailJS integration
- WhatsApp integration
- Mobile-friendly navigation
- Smooth scrolling animations
- Local storage enquiry backup

### Admin Dashboard
- Password-protected admin access
- View and manage enquiries
- Mark enquiries as read
- Delete enquiries
- Search & filter functionality
- CSV export support
- Customer reply system using EmailJS
- Statistics dashboard

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- EmailJS
- Local Storage
- Netlify Deployment
